Luma AI Agent 跨境电商 ERP 系统架构

Java + Python 协同 · LangGraph 编排 · Text-to-SQL · 向量检索

AI 工作流 (LangGraph)

  • • 建品：素材 + LLM → 标题 · 卖点 · 图片方案
  • • 诊断：Text-to-SQL + 向量检索 → 归因报告
  • • 优化：历史案例推荐 + 任务生成
  • • 复盘：优化前后对比 + 经验沉淀入库
  • • 人类介入：interrupt() 暂停等运营确认
  • • 路径回放：Checkpoint 恢复任意历史状态

Java 服务层

  • • 权限管理：RBAC + 数据范围隔离
  • • 状态机：商品 · 任务生命周期流转
  • • 任务引擎：创建 → 分配 → 执行 → 验收
  • • 指标宽表：销售 · 库存 · 广告 · 内容多维度
  • • 结果落库：AI 产出持久化 + 版本管理

数据与基础设施

  • • PostgreSQL：业务数据 · LangGraph State
  • • ClickHouse：指标宽表 · OLAP 实时查询
  • • Milvus：商品向量 · Prompt模版 · 案例检索
  • • RabbitMQ：Java ↔ Python 异步解耦
  • • MinIO：素材 · 图片 · 文件存储
